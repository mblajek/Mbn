Library for PHP and JS to do calculations with any precission, with correct approximations

project page: http://mirkles.net/n/index.php - library files avalaible there

   Pages:
      index.php - Main page - destription of library with examples, also runs unit tests
      calc.php - Calculator based on library ( http://mirkles.net/n )

   Library:
      mbn.js - JavaScript version
      mbn.php - PHP version

   Developer tools:
      mbn_test.js - Unit tests for JS version
      mbn_test.php - Unit tests for PHP version
      mbn_test_set.json - Common set of tests
      mbn_release.php - Generate slim and minified versions
